## Descripción 🏝

HTML plano que simula aplicación de lista de tareas con JavaScript y que permite añadir, completar y eliminar tareas, con actualización dinámica de la interfaz.

## Tecnologías ✒️
Para crear este proyecto he usado:
* HTML
* CSS
* SCSS
* Javascript
* Bootstrap
* jQuery
* Metodología BEM
